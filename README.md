# Rastino AI

> Building AI products for Persian speakers should not feel like a translated copy of someone else's work.

Rastino started as a simple idea:

What if Persian users could access modern AI tools through a platform designed specifically for them, instead of adapting themselves to products that were never built with them in mind?

Today, Rastino is an evolving AI platform focused on conversational AI, image generation, educational assistants, and practical AI workflows for real users.

This repository contains the public portfolio version of the project.

---

## Why I Built Rastino

Over the years I worked across multiple technical domains including:

* Machine Learning
* Deep Learning
* Data Mining
* Python Engineering
* Databases
* Web Development
* Linux Infrastructure
* Docker & Deployment

One thing became obvious:

Most AI products available to Persian users are either incomplete, poorly localized, overly expensive, or simply not designed for the way people actually use AI in Iran and Persian-speaking communities.

Rastino is my attempt to solve that problem.

## Current Features

### AI Chat

A modern multi-model chat experience designed around productivity and usability.

### AI Image Generation

Generate and manage AI-generated images directly inside the platform.

### Student Assistant

A dedicated educational environment designed around textbooks, learning workflows, and future retrieval-based study assistance.

### Administration Layer

Built-in administration capabilities for:

* User management
* Plans and subscriptions
* Usage monitoring
* Prompt management
* Platform settings
